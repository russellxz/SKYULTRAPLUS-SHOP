## ✨ Lo que obtienes
- ⚡ **Automatización total:** planes con ciclo **semanal / 15 días / mensual**, **facturas automáticas** y renovación.
- 💳 **Cobros integrados:** **PayPal** y **tarjeta con Stripe** (pagos rápidos y seguros).
- 🧾 **Facturas completas:** ver, pagar, descargar PDF y llevar histórico.
- 👑 **Admin personalizable:** cambia **nombre**, **logo**, **WhatsApp** y **URL de tu grupo** desde el panel.
- 🆘 **Soporte con tickets:** tus clientes abren y gestionan tickets.
- 👤 **Cuentas de usuario:** registro, login y **verificación por correo (SMTP)**.
- 🌓 **Modo claro & oscuro**, accesos rápidos y menú lateral.
- 💰 **Créditos (USD/MXN):**
  - Los usuarios **recargan saldo** y pagan con su balance o **pagan directo** cada factura.
  - El **admin puede abonar créditos manualmente** si un pago se realizó por fuera.
- 🔁 **Gestión de servicios:** activar, cancelar y renovar servicios según el ciclo.
- 🔐 **Confirmación de pagos:** PayPal (API/IPN) y Stripe con página de confirmación.

> **Nota sobre PayPal sin cuenta:** el pago con tarjeta como invitado depende del país y moneda de tu cuenta merchant. Para tarjetas de forma consistente, Stripe está integrado de forma nativa.

## 🧭 Flujo del cliente
1. El cliente se registra o inicia sesión.
2. Elige un **producto/plan**.
3. Paga con **PayPal**, **tarjeta (Stripe)** o **recarga créditos** y paga con saldo.
4. Se **genera la factura**, se **activa el servicio** y se **entrega la info del producto** automáticamente.
5. En cada ciclo, se **crea una nueva factura** para renovar.
